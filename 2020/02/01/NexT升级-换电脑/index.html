hexo NexT升级/换电脑

发表于 2020-02-01
用hexo搭建博客并部署到github

NexT主题升级或者换电脑最方便的办法就是重新安装一遍。先备份原来的博客文件夹，等重新安装以后把文章、图片以及修改过的地方重新改一遍，所以有了这篇，不然我也忘了自己改了哪些东西了。

更新后的版本如下：

hexo-cli: 3.1.0
node: 12.14.1
next: 7.7.0

1、显示中文

hexo 中文关键字改为了

language: zh-CN

2、贴标签

确认站点配置文件有

tag_dir: tags

确认主题配置文件有

tags: tags

3、添加打赏按钮

把收款二维码放在/next/images/下，在主题文件reward字段去掉注释，添加图片路径。

4、显示摘要

写文章时手动插入<!-- more -->

5、添加本地搜索

在博客根目录下运行下面命令，安装本地搜索插件

npm install hexo-generator-search --save

在站点配置文件中添加下面内容：

在主题文件使local_search字段生效。

6、引用站内文章

假如要引用《用hexo搭建博客并部署到github》这篇，则用这篇文章的源文件 pages.md 文件名pages替换就可以。
