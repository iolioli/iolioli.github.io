深圳市外人才引进落户流程

发表于 2019-05-28 更新于 2020-02-01 阅读次数：

落户流程

交社保一个月-通过人才测评申报-体检-办准迁证-办迁移证-落户/办身份证

前期准备资料如下：

  • 身份证正反面
  • 户口本
  • 毕业证书
  • 学信网在线学历认证有效期半年

毕业生、在职人才引进测评与申报

需要在深圳市缴纳1个月社保有接收单位，然后注册登录深圳人力资源和社会保障局-人才引进（毕业生、在职人才引进测评与申报系统），根据提示填写信息，3天左右会收到结果。通过后会收到《招用员工通知》和手机短信指标卡，记录好指标卡编号，后面要用到。

体检

去指定医院体检，深圳市拟引进市外人员体检医院名单，就近选择。医院有专门的入户体检通道，体检完毕当天下午3点就能知道结果，体检表上要盖章登记，注意不要把体检编号写错，医院会把体检合格结果上传到人社局系统。大概2天就会录入。

在深圳市不动产登记信息查询自助生成无房证明。

户籍迁入

在深圳市公安局-户籍迁入—-招调工户籍迁入。单位没有集体户的话户口选择派出所代管。接着按要求填写信息上传证件，准迁证选择邮寄方式或自取。前面提到的资料都需要上传图片。这一步最好在电脑上操作，微信里面太费劲了。大约一周准迁证就下来了，带准迁证到户籍地办理迁移证，可让家人代办。

办理落户

迁移证办好后去公安局办理落户，先在深圳市公安局公众号预约，需要准备电子照片和回执。

（回执就是一个公安系统的照片编号，可以在支付宝搜索“证件照带回执”自助办理，后面办理金融社保卡也需要照片回执。深圳这样的城市竟然在行政大厅没有自助拍照设备，难免让人怀疑），当场办结，身份证大概10个工作日办好，可以选择邮寄。

申请租房和生活补贴

在人力资源和社会保障局官网-新引进人才租房和生活补贴，按照要求填写材料，注意毕业日期要和毕业证书上的日期一致。

银行卡只要是一类卡就行，外地卡也可，可以不是社保卡。

这月申请的下个月会汇总，大概一个半月后公示，公示5个工作日结束后大概半个月会收到补贴。整个流程下来近3个月。

关于社保

如果人不在深圳市上班可以找挂靠交社保，社保挂靠的话外地户籍交医疗保险可以交三档，算下来是五百多，医疗保险一档是九百多，注意别被坑太多。可选然后自己申请落户，没必要花钱让代理办。整个过程中就体检、到派出所落户和办新身份证才需要出门，一共2次，其他流程都是网上进行的。
