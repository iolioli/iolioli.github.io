文科生也能学的Python，Linux入门资料

发表于 2017-04-06 更新于 2020-02-01 阅读次数：

我经常感叹市面上没有好的学习资料，尤其是计算机方面，在没有接触过计算机的时候，看到的尽是“xx开发实战”之类的书，怎么可能学得下去。对于没学过的人来说，面对的可不只是一门计算机语言，还有很多计算机方面的其他问题。面向专业人员的书和教程多如牛毛，但是真正面向零基础同学的资料却寥寥无几。

我有一次搜到一份Linux的英文学习资料，发现都能看懂，感觉曾经神秘的Linux也不是那么恐怖嘛，然后发现有中文版，可是中文版书名是”linux命令行大全”，看到这样的书名还以为是什么查询之类的工具书吧。

然后你再再看看英文版的The Linux Command Line by William E. Shotts

不过我找到一个不错的中文版：https://billie66.github.io/TLCL/book/

看过上面这本书以后，我算是对计算机有了一些认识了，即使你是文科生上面的内容也完全看得懂，不需要你的数学好，初中的数学知识就足够了。

实话说虽然大学里教过C语言，但是当时根本不知道老师说了些什么，倒是因为自己学习谷歌搜索技巧，装了Linux系统，学着搭建shadowsocks服务，搭建个人的WordPress博客慢慢地积累了一些知识。有了学习编程语言的兴趣。所以我深知开始学习一门计算机语言的茫然。

Python非常适合入门，容易理解，推荐下面的资料
automate the boring stuff with Python
不多说，能看懂的才是好资料。
