《简单的逻辑学》《一课经济学》摘录

发表于 2017-04-16 更新于 2020-02-01 阅读次数：

观念的来源

  1. 大脑中的每个观念最终都源于对事物的描摹，而真实的事物却独立于观念之外
  2. 给观念找到合适的语言并不是一个自发的过程，这一工作常常充满挑战性，我们不是都有词不达意的时候吗？
  3. 客观存在的事物-事物在大脑中的反映-为其创造语言

有效沟通

  1. 语言和观念的匹配度越高，沟通就越清晰有效
  2. 不要想当然地认为你的听众会领悟你没有直接表达的意思
  3. 不要对外行人使用业内行话
  4. 沟通的关键是理解
  5. 尽量直抒胸臆，降低听众对你要表达的意思产生误解的可能性

经济学

  1. 被政治选中的接受优待的一群人的各种政治可能性，明眼人一看便知
  2. 最低工资法剥夺了社会享有这部分人（指原来工资不足最低工资的那些人）的服务所创造的价值，也剥夺了这些人凭自力更生而拥有的独立与自尊
  3. 我们都习惯于用金钱来衡量自己的收入和财富，这种惯性十分强烈（比如涨工资）
  4. 通货膨胀有如自我暗示、催眠术、麻醉剂可以减轻手术时的痛苦，通货膨胀也像鸦片，吸上一口就觉得自己无所不能
  5. 通货膨胀等于变相征税
