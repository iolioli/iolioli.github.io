台湾景点猫空缆车龙坑官网预约

发表于 2017-12-11

猫空缆车座位预约

翻墙后登录猫空缆车官网

  1. 点击要预约的日期后，可以看到不同时刻的剩余预约人数。
  2. 选择普通车厢或者水晶车厢，选搭乘时间
  3. 填写申请人资料，需要填通行证、国籍、手机或电话、预约人数（需要注意6岁以下儿童虽然不收费，但是也要预约作为。假如一个大人带一个6岁以下的孩子就需要预约2个座位。）、邮箱，填写后点击“下一步”。
  4. 保存或打印预约单。因为官网原因大通证号码显示不全没有关系，最主要的是条形码。

龙坑 南仁山生态保护区拼团

翻墙后登录龙坑南仁山生态保护区环境教育系统，进来后就是拼团页面

  1. 选择龙坑上午或下午，选拼团人多的这样拼团成功率会高一点，超过10人拼团就成功了。
  2. 填写资料后点“送出”
  3. 填写每个访客的姓名通行证资料，送出
  4. 确认资料无误后送出，会生成一个序号，这个序号需要记下来。等拼团人数超过10人会通知你付款。

故宫定时导览预约

翻墙后登陆故宫博物院线上报名系统，选择“中文定時導覽”-“我要报名”-“申请”-选择时间（每天4个时间，每个时间段15个名额：9:30/10:00/14:30/16:00) 导览全程免费，大概1个半小时到2个小时左右。可以提前三个月报名，额满截止。可以看到剩余名额。

  1. 选好日期后，填写资料，证件选护照，填大通证号码。
  2. 确认信息，打印或截图即可。
