一种计算机语言包含了各种语法、规则、语法规则的解释器、相关的文档等。人和人的交流也是如此困难，何况和计算机，所以你的命令一定要符合 Python 的语法规则，否则就会出错。假如你漏掉了一个引号就会出错

凡是不能让解释器顺利翻译，导致出错的情况都叫 bug，找出错误叫 debug，学习任意一种计算机语言这两个词都将伴随你。

这一点是我最不理解的，为什么这时候还要用电话这种落后的方式。难道政府部门的领导从来没有网购过吗？客服软件相比电话至少有这些优点：

台湾10日环岛旅游攻略

发表于 2017-12-09

马蜂窝和穷游的问答板块上有各种各样的提问，其中最多的就是行程规划，虽然大数据人工智能发展了这么久，可是还没有一个好用的可以帮助规划行程的工具，我以台湾环岛旅游为例总结一下出国旅游如何规划行程。

挑选景点

一般出发前我们首先会了解有哪些景点，通过浏览别人的游记等选出了自己中意的一些景点，比如挑选出了101大楼、台北故宫博物院、清境农场、太鲁阁、清水断崖、九份、十分、后壁湖、鹅銮鼻、日月潭、阿里山、平溪线、猫空缆车等等。

城市归类

我们选了一大堆自己中意的景点然后把它们按照城市分类，城市内的交通不需要担心一般是地铁、公交出租车。分类后

  1. 101大楼、台北故宫博物院、猫空缆车属于台北市内
  2. 九份、十分、平溪线属于新北市，离台北不远
  3. 后壁湖、鹅銮鼻在垦丁
  4. 太鲁阁在花莲

这差不多是台湾最受欢迎的几个旅游地点了。

城市切换

城市之间切换是行程规划中最麻烦的，因为我们要把桃园机场、台北、新北、垦丁、花莲这几个城市连起来，坐什么车，还要考虑如何合理地分配时间，这就需要查每一段行程需要的时间。

要想知道时间就要了解台湾有什么交通，查询后发现台湾有高铁、台铁（类似我们的火车）。但是高铁只有南港-左营（高雄市）这一段才有

台铁是环整个岛都有，了解这些后就可以查询：

  1. 机场到台北坐什么车，用时多久。一般机场到市内都有机场巴士。
  2. 新北市的火车站叫瑞芳站，台北到新北可以坐台铁，到了瑞芳站就可以坐公交或者出租到其他地方了。
  3. 去垦丁不方便，一般都是台北站乘高铁到左营站，左营站就可以乘垦丁快线到垦丁
  4. 垦丁和花莲之间没有直达的台铁，必须经过枋寮站中转。另外通过看游记还可以拼车。
  5. 花莲到瑞芳、花莲都台北都有直达

所以具体的安排可以有很多种

  1. 桃园机场-桃园高铁站-左营站-垦丁-枋寮站-花莲-瑞芳-台北-桃园机场
  2. 桃园机场-台北-花莲-枋寮站-垦丁-左营站-台北-瑞芳-台北-桃园机场

到一下站坐什么车，用时大概多久都需要查清楚，路线规划的差不多以后再买票。

win10 UEFI+GPT模式安装：解决磁盘包含MBR分区

发表于 2017-11-24 更新于 2019-04-17

本文将整理安装 win10 遇到的一些典型问题。

制作启动盘

在MSDN我告诉你下载win0最新版，下载U盘启动制作软件rufus，巨好用不到1M的大小，用默认方式制作就好。不要用老毛桃、博士装机、winpe等，装完就是一堆360、腾讯全家桶。

UEFI legacy模式， MBR GPT分区方式介绍

UEFI、BIOS是引导系统安装的程序，UEFI可以看做BIOS的新一代，将取代BIOS，所以如果电脑支持uefi引导最好通过uefi安装，现在的电脑基本都支持了。MBR、GPT表示磁盘的分区方式，GPT设计来取代MBR。 如果你的电脑支持uefi但你就想通过老式BIOS来安装，那就要将boot mode 切换为legacy。

但是如果你的电脑支持UEFI安装，那默认会按照UEFI方式安装，如果你强行以legacy模式安装的话就会出现

Windows 无法安装到这个磁盘。选中的磁盘具有MBR分区表。在 EFI 系统上，Windows 只能安装到 GPT 磁盘

所以支持UEFI就用UEFI安装不要浪费了，用UEFI模式安装，且把磁盘改成gpt分区。在选语言的安装界面按shift+F10进入命令行，如图

输入diskpart回车，就会自动进入Diskpart模式.
输入list disk，列出磁盘
输入select disk 0，选中那个磁盘（根据你的实际情况，不一定是disk 0）
输入clean，清除MBR分区
输入convert gpt，转为gpt分区
输入exit退出Diskpart
输入exit退出，这时返回了安装界面

这样就可以安装了。

一直在准备就绪

第一次安装win10在这里的时间特别长，根据电脑配置情况不同，我用一台老笔记本安装足足用了30分钟，第一次安装以为出问题了就强制重启了，第二次没管先去洗澡了，洗完后发现到了下一步。

这一步一定要耐心一点，不然可能导致

计算机意外地重新启动或遇到错误。Windows 安装无法继续。若要安装Windows，请单击“确定”重新启动计算机，然后重新启动安装

U盘恢复

用 rufus 制作的系统U盘删除文件就可以了。有的人是淘宝购买的系统盘，这种会留下一个CD驱动器：系统U盘，格式化不了，这种情况最简单的办法是下载chipgenius来检测U盘的flash型号，然后下载对应flash型号的量产工具即可修复。

连岳唯二推荐的文学书（另外一本是了不起的盖茨比），作为一本文学书情节设定本身已经惊世骇俗，再加上我多多少少了解一点99年以前的历史，而且读的时候谷歌刚刚被封，新一轮xx愈演愈烈，带入感太过强烈仿佛自己就在书中冒险，真的是浑身颤抖并且小心翼翼地地读完的。

如何准备日语一级考试(JLPT N1)

发表于 2017-10-03

备考资料

备考资料只需要历年真题和《耳から覚える日本語能力試験 》这两本就够了。《耳から覚える日本語能力試験 》又分为语法和词汇，很多人把这本书浪费了，仅仅用它来背语法、背单词。

这本书正确的用法是来练习听力

练习听力的同时顺便理解语法

听力才是决定你能不能（高分）通过的关键，如果你能听懂了，那阅读自然不在话下。

练习听力

该书正确的使用方法：

先不要看原文一句一句地听，比如第一个语法是や否や，录音会先读语法点再读例句，仔细听例句是怎么断句的，听的时候就要注意这个语法是怎么接续的，例句是

空が暗くなるや否や大粒の雨が降りはじめた

你会听到おおつぶのあめ，如果你不知道「大粒」这个词，那你听的时候至少要转化成「おおつぶの雨」，你要能把おおつぶ听出来，要把每一个音（假名）都能听出来。这就是在训练听力呀。不要为了练习听力而听，不要为了背单词而背单词，这本书就是为了让你边听、边理解、边记。

把每一个例句完全搞懂，录音放在手机里随时听，一个单元大概12个例句也就是12句话，4分钟左右，设置单元循环播放，播放几次后，一读到语法点你就想起了例句是啥，你不需要刻意背就记住了。背单词也是这样。

另外再推荐这本书，外研社出的，有中文解释

国内所有的能力考试语法书都离不开这本，包括上面耳から覚える，上面那本很多例句都是要么把这本书的例句的前半句改一下，要么把后半句改一下，有的干脆照搬这本书的例句。

N1语法剖析

语法要死记硬背吗？N1的语法是什么？

N1的语法偏书面化一点，几乎所有的N1语法你都可以用已经学过的语法表达出来，也就是说只是表达方式的不同，或者在表达的意思上有轻微的不同，这个时候是为了让表达更精确，更接近于你想表达的意思。

在干货开始之前，请确认下面这些的基本用法你已经掌握

から　まで
こそ
こと　　もの　
ただ　のみ
である　ではない
と　という
となる　とする
とも　ども
になる　にする
は　も　で/ても　
ばかり　ところ
条件形ば 意志形 可能形

N1的语法就是在这些基本用法上面的一点点变形。不熟悉的请看可以阅读的日语自学手册。

ない之外的未然形

ない这种未然形我们已经很熟悉了，除此之外的未然形还有ず　ざる　まい

よるーよらず

べからざる　べからず

まい比较特殊，是在动词原形后面加まい，比如 ある的这种未然形是 あるまい（更多解释请度娘）

另外一个技巧是

把假名转化成汉字

很多语法通常写成假名，但是这些假名是有对应的汉字的，看了汉字你就恍然大悟了，比如N1语法

いかん いかんによらず

写成这样

如何　如何によらず

是不就明白多了，同样的很多用法都是手到擒来，比如

かたわら　傍ら
きらいがある　嫌いがある
にかかわる　に関わる
にたえない　に耐えない
まみれ　塗れ
ゆえ（に）　故（に）
ごとき　如き（如く）
にあたらない　に当たらない
にひきかえ　に引き換え
にもまして　にも況して
をよそに　を余所に
しまつだ　始末だ
てやまない　て止まない
ないではおかない　ないでは置かない
にかたくない　に難くない
ずにはすまない　ずには済まない
そばから　側から

这些用法只要你知道汉字是什么意思，一目了然啊，相当于就记一个单词，根本不需要花多大功夫，何况很多你已经认识了。

有一些编在语法书上完全就是凑数的，比如

限りだ
に関わる
始末だ
に耐えない

这些就是一个普通名词或动词。

有一些看似很长的只要“转个弯”就行了，拆开相当于是两个否定

ないもの でもない
ないでは おかない
ずには おかない

有的完全是误导人，比如からある，看一个例句

その男は三十キロからある荷物をひょうと肩に担いっだ
这句话断句应该是这样的

その男は三十キロから ある荷物をひょうと肩に担いっだ
我们知道から表示“起点”，而ある日、ある時、ある人物就相当于某日某时某人，这样的表达应该早就见过了，所以这句话的字面意思是“那个男人把三十斤起的 某物 抗在了肩上”。

除去这些简单变形的语法，真正需要理解记忆的就剩下

がてら
が早いか
や否や
っぱなし
なり…

也没多少了。

如何有效地看日剧动漫来学日语

发表于 2017-10-01

快速学语法

教材在我看来就是文章+录音+语法讲解。

传统教学里老师就是对照课文遇到语法点讲解一下，布置作业辅导一下，通过大量的习题来巩固语法。对照教材学习其实是从文章中揪出语法来，聪明的同学会先看一遍语法讲解，再跟着老师的讲解过一遍。可是等到一学期一本书学完了发现没学多少语法，几年下来也没学多少。

其实我们完全可以快速地先学习语法。然后用大量的阅读、听力材料来验证、巩固这些语法，把老师给替换掉。

积累单词 理解语法

日剧动漫是特别好的材料，完全代替教材文章+录音的功能，有剧本、有剧情、有多个人的发音、有场景比教材丰富多了。既然是学习就要抱着学习的态度，而不是娱乐的态度。

争取把每一句话都听懂，这就需要你遇到单词要查，出现新的语法也要查，跟着教材学不也是这样做的吗。

一句话听完以后再跟着模仿剧中人物的口吻对着siri说一遍，安卓可以用simeji语音输入法，看它能不能识别你说的话，把识别后的文本顺便保存在文档里，这就是你的语料库

三步走

  1. 听懂（查单词，查语法）
  2. 说（模仿，练习口语）
  3. 记录（语音识别，验证发音）作为语料库

举例
下面是日剧最完美的离婚中的台词

頭がいいとか、知識があるとかより、人を元気にできることの方がよっぽど価値がある。素直でまっすぐに生きていてる方が周りを幸せにする。

看到不会的单词就查，比如よっぽど
记忆模糊的语法重新查一下手册，就这句里包含的语法有

  • とか
  • より　ほう　作比较
  • 两个形容词并列该怎么表达
  • にする
  • こと

把不明白的查过以后，语音识别转换成文本，你要是说的不准可识别不了哦，这就相当于给你的反馈，发音是不是正确，老师做的事情也不过如此。

练习听力和口语

选一部你喜欢的现代剧认认真真的按照这样的节奏学下去，火影忍者这样的动漫是不行的，一开始会感觉很困难，万事开头难嘛，这一步得走，后面就顺利多了，大概两个月吧。

接下来就是不看日文字幕了，因为这时候词汇量也有了，基本语法也掌握了，慢慢地听写，然后看你的句子顺不顺，对照中文字幕看意思对不对。

我当时听了金田一少年事件薄，很多话现在还记着呢，我没有刻意地去背，不过一遍又一遍地听所、一遍一遍地复读，所以记住了。

この手紙をあなたが読む頃、私はもう生きてはいないでしょ、昨日あの三人の話を聞いてから、私の心はこの顔と同じように醜くなりつつあります。心まで変わり果てる前に、命を絶つつもりです。不思議ね今は悲しいことも後悔もないの、手を伸ばせば天国に触れられるように気がするから。

注：这是自杀的女孩给男朋友的遗书，女孩被另外三个人毁容了，所以自杀了。男朋友为她复仇的故事。可是男的并没有看到这封信。
